# TikTok Story Viewer Using Python (with Requests)

📢 Content & updates via Telegram: [@z_0_g](https://t.me/z_0_g)

## Features

 - Fetch story details including:
  - Video/image URL
  - Cover image
  - Heart count
  - View count
  - Timestamp
